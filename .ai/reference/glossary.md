# Glossary

Updated: 2026-03-08
Owner: @ivans

## Terms
- `UoW` (Unit of Work): abstraction that coordinates repository access and commits.
- `DTO`: Data Transfer Object used for API/application contracts.
- `Result<T>`: success/failure wrapper for expected business outcomes.
- `RTK Query`: Redux Toolkit data-fetching and cache layer.
- `baseApi`: shared RTK Query API instance where endpoints are injected.
- `ADR`: Architecture Decision Record documenting important design choices.
- `AAA`: Arrange, Act, Assert testing structure.
